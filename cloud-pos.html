Introducing the PoS

The Cloud POS Application is the core tool used by employees, cashiers, and managers to handle transactions, manage orders, and process payments in real time. It’s designed to be touch-friendly, fast, and user-friendly, with full support for multiple languages (Arabic and English), dynamic product loading, and multi-branch operations.

The following manual focuses on the POS terminal interface only, where employees interact with products, orders, and payments.

Overview

This document explains how to use the Cloud POS app efficiently and guides you through every available function in the system.

Getting Started

This section introduces how to access and log in to the Cloud POS app from supported devices like desktop stations or Android terminals.

Accessing the PoS App

To get started, launch the Cloud POS desktop application or the Android terminal app provided by your restaurant’s system administrator.

Login into the PoS App

After launching the app:

  • Enter your assigned passcode.
  • Select your employee name from the list.
  • Once authenticated, a prompt will appear to start a new shift.
  • After confirming, you will be taken directly to the categories grid screen, which serves as the main order interface.

Opening a new shift will be explained in detail in the “Shifts” section.

The PoS App Interface

This section describes the user interface and functionality of the POS terminal, including how to navigate and use key features.

UI Overview

The POS interface is designed to be intuitive and fast. It is divided into several panels: product grid, cart area, total summary, and action buttons such as payment, hold, or void.

FAQs

What is Smart Prints?

A cloud-based retail system combining POS, kiosks, inventory, menu, and service tools into one integrated platform.

Can I start with just one service?

Yes, you can begin with any module and add more later. All services are modular and plug-and-play.

Is Smart Prints fully cloud hosted?

Yes. You can manage all services from anywhere using a browser-based dashboard.

Do I need technical skills to configure it?

No. We provide full setup wizards and guides, and most operations are no-code.

Is there a support team I can contact?

Yes, support is available via the admin panel or through the contact links in the Resources section.

How often is documentation updated?

Documentation is reviewed with every feature update or platform improvement. You’ll find timestamps at the end of major pages.
